# M4_AE6_ABPRO-Ejercicio-grupal

Participación en la actividad

Este trabajo fue desarrollado de manera individual, realizando todas las etapas del proceso por mi cuenta.

Me encargué de:

- Analizar los requerimientos del sistema.

- Diseñar la estructura de datos para almacenar el inventario.

- Crear y organizar las funciones necesarias para leer, escribir, modificar y eliminar información del archivo inventario.txt.

- Implementar correctamente los modos de apertura de archivos:

- r (lectura)
- w (escritura)
- a (append)
- r+ (lectura y escritura)

- Implementar las funciones para buscar productos, modificar nombres, eliminar registros y visualizar atributos del archivo.

- Diseñar el menú interactivo para ejecutar las funcionalidades requeridas.

- Probar cada parte del programa asegurando que los datos se escribieran y leyeran correctamente del archivo.
